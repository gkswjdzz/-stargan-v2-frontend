## Build
### Prerequisite
```bsh
$ npm install
```

### Run on terminal
```bsh
$ node server.js
```

## Dockerize
### build dockerfile
```bsh
docker build -t <image_name> .
```

### run docker image
```bsh
docker run -p 8008:8008 -it <image_name>
```
